#### Description:

My cs50 project is an ecommerce website and its title is bike heaven.

1. [Home](/):
First of  all there is homepage in my website which have 3 sections. One is showcase in which there is background image on which some text is written.second one is trending products where we display all our trending products and third one is testimonails where we display some testimonail using cards and at the end of the homepage this is a footer which contain links to other pages of the website and all these sections in presented in index.html.  

2. [Registeration](/register):
Secondly we have registeration page in the website. On registeration page we have 3 input fields named as username, password,confirm password and a checkbox for accepting terms and conditions.On three input field i put  some condtions for security purpose and these conditions are: 1: username must be unique, password minimium length will 8 characters and password must contain some special character as well as numeric character. 2: All the field should be must filled. 3: password and confirm password must be similar. if any of the condition is not satisfy then an alert message will be display to the user related to the condition which is not satisfy.registeration page is present in templates directory name as registeration.html

3. [Login](/login):
Third one is login page. On login page of the website we have two input fields and one submit/login page.two input field is named as username and password and on both input we have apply some condtions for security purposes.when the user fill the input we will check it in the database either the person have any account or not after confirmation we give permission and the user has been login.

After loging in the user have the permission to make thier profile and to buy and to add to cart any of the product he want.

4. [Edit profile](/edit):
Forth one is to make your profile on edit_profile which is present in edit_profile.html. on this page we make 6 field which are firstname,lastname,email,phone_number,address,cnic and file field for profile image and can make or change profile for here.

5. [View profile](/profile):
Fifth one view profile where we will display the information that you entered in the edit profile and you can also change these data by click edit button in your profile. this page is presented  in the project directory by the name profile.html

6. [Products](/products):
sixth one is products page on the website where all the products are displayed for buying including the trending products. By clicking on product card you will be redirected to another page which is product description page where you can buy or add to card any of the product.after clicking on buy we will ask you for confirmation of buying product if you click yes then your ordered will be placed and you ordered product will be showed on purchases page.

7. [Purchases](/purchases):
Seventh page is purchases page of the website where all the products will be displayed that you buyed

8. [Cart](/cart):
Eight one is cart page where all the products are displayed which you added to cart but the one extra thing is that you can remove any product that you added to the cart.
